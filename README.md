Dragon Book 2nd 
===============

C++ implementation for Compilers Principles, Techniques, &amp; Tools 2nd (i.e. 编译原理)


### On Linux
- to compile : 
	`g++ -std=c++11 -Wall -o run_me some_exercise.cpp`
- to run :
	`./run_me`

### On Mac
- same as that on Linux.

### On Windows
- use VS(2012+) Command Prompt:
 - `cl /EHsc some_exercise.cpp`
 - `some_exercise.exe`
